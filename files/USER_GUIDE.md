## С чего начать

1. Найдите бота в Telegram (уточните username у администратора или в объявлениях)
2. Нажмите **Start** или отправьте `/start`
3. Напишите на естественном языке что вы хотите — например: `Расписание ИСС-б-о-22-3`

Никакой регистрации, никаких паролей. Бот запомнит вас автоматически.

## Команды бота

| Команда | Что делает |
|---|---|
| `/start` | Приветствие и краткая справка |
| `/help` | Полная справка по возможностям |
| `/miniapp` | Открыть расписание в приложении |
| `/limit` | Посмотреть остаток лимита запросов |
| `/roles` | Посмотреть свои роли и привилегии |
| `/support <текст>` | Написать в поддержку |
| `/suggest <текст>` | Предложить идею или улучшение |
| `/about` | Информация о боте и ссылки |

## Как спрашивать расписание

Бот понимает **обычный русский язык**. Не нужно учить специальный синтаксис — пишите так, как написали бы человеку.

> [!TIP]
> Название группы можно писать в любом формате. Все варианты ниже — одно и то же:
> - `ИСС-б-о-22-3`
> - `исс б о 22 3`
> - `ISS-b-o-22-3`
> - `аис25` → автоматически разворачивается в `аис-б-о-25`

## Расписание группы

### По умолчанию — текущая неделя

```
Расписание ИСС-б-о-22-3
```

### Другие временные выражения

```
Что у группы АИС25 завтра?
Пары ИСС-22 на следующей неделе
Расписание МАТ-б-о-23-1 в понедельник
Что сейчас у группы АИС-б-о-25-1?
Есть ли пары у ИСС-22-3 в пятницу?
```

### Навигация по расписанию

Когда расписание большое, бот разбивает его на дни с кнопками навигации:

```
◀ Пред.  [День 3 из 6]  След. ▶
```

Нажмите кнопки чтобы листать дни.

### Оценка ответа

После каждого ответа бот показывает кнопки 👍 и 👎. Нажмите — это помогает улучшать систему.

## Расписание преподавателя

```
Расписание Подзолко на эту неделю
Где Иванов сейчас?
Что ведёт Щербина завтра?
Расписание Иванова И.И. в среду
Где сейчас Безпалько?
```

> [!NOTE]
> Фамилию можно писать в любом падеже — бот поймёт «Иванова», «Иванову», «Ивановым».

### «Где преподаватель прямо сейчас?»

```
Где Подзолко?
Где сейчас Иванов?
Где Щербина через 10 минут?
```

Бот покажет: в какой аудитории идёт пара, какая дисциплина, до какого времени.

## Поиск

Если вы не знаете точное название группы или фамилию преподавателя:

```
Найди группу ИСС
Найди Петрова
Найди аудиторию 405
Список институтов
Покажи институт информационных технологий
```

Бот покажет совпадения в виде кнопок — нажмите нужный вариант.

### Если совпадений несколько

Когда бот находит несколько похожих вариантов, он спрашивает уточнение:

```
🔍 Найдено несколько совпадений:
[ИСС-б-о-22-1  •  ИИТ]
[ИСС-б-о-22-2  •  ИИТ]
[ИСС-б-о-22-3  •  ИИТ]
[Показать ещё ▶]
```

Нажмите нужную группу, и бот покажет расписание.

## В групповом чате

Бот работает в групповых чатах, но **не слушает общую переписку** — только прямые обращения.

### Как обратиться к боту в группе

**Способ 1 — Упоминание:**
```
@bot_username расписание ИСС-б-о-22-3
```

**Способ 2 — Ответ на сообщение бота:**
Нажмите Reply на любое сообщение бота, напишите вопрос.

**Способ 3 — Команды:**
Все команды (`/help`, `/miniapp` и т.д.) работают в группах без упоминания.

> [!NOTE]
> Лимит запросов в группе — общий для всего чата (3–5 запросов за 7 часов), не индивидуальный. Спланируйте использование вместе с участниками.

## Telegram Mini App

Mini App — полноценное приложение расписания прямо в Telegram. Откройте его через `/miniapp` или кнопку **«📅 Открыть расписание»**.

### Что умеет Mini App

| Вкладка | Возможности |
|---|---|
| **📅 Расписание** | Расписание по группе, преподавателю или аудитории |
| **🚪 Аудитории** | Свободные аудитории прямо сейчас или в нужное время |
| **⭐ Избранное** | Сохранённые группы, преподаватели, аудитории |
| **👤 Профиль** | Настройки, тема, остаток лимита |

### Как пользоваться расписанием

1. Выберите тип: **Группа** / **Преподаватель** / **Аудитория**
2. Начните вводить название — появится поиск с подсказками
3. Выберите из списка — увидите расписание на неделю
4. Листайте недели стрелками ← →

### Избранное

Добавьте часто используемые группы и преподавателей в избранное — они появятся на вкладке ⭐ для быстрого доступа.

### Настройки (вкладка 👤)

- **Тема**: светлая / тёмная / автоматически
- **Формат времени**: 24ч или 12ч
- **Компактный вид**: уменьшенный шрифт для большего расписания
- **Акцентный цвет**: персонализация интерфейса

## Лимиты запросов

Бот использует AI для обработки запросов, и у него есть ограничения.

### Как работают лимиты

- **3 запроса за 7 часов** — в личном чате (стандартный аккаунт)
- **3–5 запросов за 7 часов** — в групповых чатах (общий лимит на чат)
- Лимит сбрасывается через 7 часов автоматически

### Что не считается за запрос

Эти команды всегда бесплатны:
`/start`, `/help`, `/miniapp`, `/roles`, `/limit`, `/support`, `/suggest`, `/about`

### Проверить остаток

```
/limit
```

Бот покажет:
```
📊 Лимит личных запросов

🟢 [████░░░░] 2/3

✅ Использовано: 2
💬 Осталось: 1
🏁 Максимум: 3
🔄 Сброс через: 4ч 15м
```

### Лимит исчерпан — что делать?

- Воспользуйтесь **Mini App** — там лимит не нужен
- Подождите сброса (отсчёт покажет `/limit`)
- Напишите администратору для увеличения лимита

## Часто задаваемые вопросы

### ❓ Бот не отвечает на мои сообщения в группе

В групповом чате бот отвечает только на прямые обращения. Напишите `@имя_бота ваш вопрос` или ответьте на его сообщение (Reply).

### ❓ Бот говорит «расписание не найдено» для моей группы

Возможные причины:
- Название группы написано с ошибкой. Попробуйте `/suggest ИСС` или `Найди группу ИСС`
- Расписание ещё не загружено (скрапер обновляет данные раз в час)
- Группа существует, но у неё нет расписания в eCampus

### ❓ Расписание показывает старые данные

Скрапер обновляет данные раз в час. Если расписание в eCampus изменилось только что — подождите до следующего обновления или напишите в `/support`.

### ❓ Запрос понят неверно

Попробуйте перефразировать. Примеры:
- Вместо «ИСС 3» → «группа ИСС-б-о-22-3»
- Вместо «Петров» (может быть студент) → «преподаватель Петров» или «расписание Петрова»

### ❓ Почему лимит 3 запроса, а не больше?

Каждый запрос к боту использует ChatGPT, что стоит ресурсов. Лимит защищает систему от перегрузки. Вы можете использовать Mini App — там лимита нет.

### ❓ Как добавить бота в свою учебную группу?

Добавьте его в Telegram-группу как обычного участника. Бот автоматически отправит приветствие с инструкцией.

### ❓ Как узнать свой Telegram ID?

Напишите `/roles` — бот покажет ваш username или `tg:ID`.
Или напишите боту [@userinfobot](https://t.me/userinfobot).
